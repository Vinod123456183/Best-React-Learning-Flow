### 🔹 Great Free APIs for Frontend Projects

**1. DummyJSON (Fake Data API)**
Use it for *products, recipes, posts, cards, tags, users, etc.* — perfect for a Recipe Gallery, Blog, Travel Gallery, or Product Landing Page. ([DummyJSON][1])

* Example endpoints you can fetch in React:

  * `https://dummyjson.com/recipes` (recipe data) ([DummyJSON][2])
  * `https://dummyjson.com/products` (product info) ([DummyJSON][1])
  * `https://dummyjson.com/posts` (blog posts) ([DummyJSON][1])

**2. Meal Recipe APIs (Optional Extra)**

* Spoonacular or TheMealDB — you can fetch real recipe details (may require API key with free tier) ([GeeksforGeeks][3])

**3. Other Free APIs for Fun Data (Optional)**

* Star Wars API (SWAPI) — great for demo cards or fun movie/character data ([GitHub][4])
* Fake Store API — product items and categories for ecommerce style projects ([Rattibha][5])
